# Contributing

Thanks for your interest in claw-brain! Contributions are welcome.

## How to contribute

1. Fork the repo
2. Create a branch (`git checkout -b my-change`)
3. Make your changes and commit
4. Push and open a PR

## What's welcome

- New slash commands (`.claude/commands/`)
- Improvements to existing commands
- Documentation fixes and clarifications
- Bug fixes

## Philosophy

Keep it minimal. No bloat, no unnecessary abstractions. This is a starter template — every addition should earn its place.

## Questions?

Open an [issue](https://github.com/Lanzelot1/claw-brain/issues) to discuss ideas before building anything large.
